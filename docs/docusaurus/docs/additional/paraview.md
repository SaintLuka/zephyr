# ParaView

[ParaView](https://www.paraview.org/) — кроссплатформенная среда для научной визуализации с открытым исходным кодом, предназначена для анализа и визуализации больших наборов данных (в том числе распределённых на кластерах). Программа создана на  основе библиотеки [VTK](https://vtk.org/) (Visualization Toolkit), позволяет интерактивно исследовать результаты численного моделирования через графический интерфейс или скрипты на Python. ParaView поддерживает широкий спектр операций: от построения изоповерхностей, срезов и векторных полей до объёмного рендеринга и анимации.
