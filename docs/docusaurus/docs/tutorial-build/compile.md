# Компиляция

## Структура проекта

Рекомендуемая структура проекта:
```
Zephyr/
 ├─ build/
 ├─ debug/
 └─ zephyr-src/
     ├─ .git/
     ├─ docs/
     ├─ problems/
     ├─ tests/
     ├─ zephyr/
     ├─ .gitignore
     ├─ CMakeLists.txt
     └─ README.md
```
Исходные файлы проекта расположены в директории `zephyr-src`, к ней же подключен git.
Сборку проекта следует выполнять вне этой папки (в `build` или `debug`).  Если сборка
осуществляется внутри `zephyr-src`, необходимо убедиться, что мусор от компиляции
не добавляется в git. Все файлы компиляции, а также конфигурации, созданные IDE, следует
добавить в файл .gitignore.
